# Spiritless

**SPIRITLESS**, _adjective_

**1.** Destitute of spirits; wanting animation; wanting cheerfulness; dejected; depressed.

**2.** Destitute of vigor; wanting life, courage or fire; as a _spiritless_ slave.

A man so faint, so _spiritless_ so dull, so dead in look--

**3.** Having no breath; extinct; dead.
