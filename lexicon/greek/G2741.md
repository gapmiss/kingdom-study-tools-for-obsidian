# G2741 καυσόω

## kausóō

_(kow-so'-o | kav-SOH-oh | kaf-SOH-oh)_

### Definition

from G2740; to set on fire; fervent heat; 

- to burn up, set fire to
- to suffer with feverish burning, be parched with fever
