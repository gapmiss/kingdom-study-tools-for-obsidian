# G2431 ἱλαρός

## hilarós

_(hil-ar-os' | hee-la-ROSE | ee-la-ROSE)_

### Definition

from the same as G2436; propitious or merry ("hilarious"), i.e., prompt or willing; cheerful; 

- cheerful, joyous, prompt to do anything
