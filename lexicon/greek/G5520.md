# G5520 χολάω

## choláō

_(khol-ah'-o | hoh-LA-oh | hoh-LA-oh)_

### Definition

from G5521; to be bilious, i.e., (by implication) irritable (enraged, "choleric"); be angry; 

- to be atrabilious
- to be mad
- to be angry, enraged
