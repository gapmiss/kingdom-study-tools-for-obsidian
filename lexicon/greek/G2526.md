# G2526 καθό

## kathó

_(kath-o' | ka-THOH | ka-THOH)_

### Definition

from G2596 and G3739; according to which thing, i.e., precisely as, in proportion as; according to that, (inasmuch) as; 

- according to what
  - as
  - according as, in so far as, so far forth as
