# G3436 μολυσμός

## molysmós

_(mol-oos-mos' | moh-loo-SMOSE | moh-lyoo-SMOSE)_

### Definition

from G3435; a stain; i.e., (figuratively) immorality; filthiness; 

- defilement, an action by which anything is defiled
