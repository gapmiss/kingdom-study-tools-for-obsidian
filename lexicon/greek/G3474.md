# G3474 μωρός

## mōrós

_(mo-ros' | moh-ROSE | moh-ROSE)_

### Definition

probably from the base of G3466; dull or stupid (as if shut up), i.e., heedless, (morally) blockhead, (apparently) absurd; fool(-ish), X foolishness; 

- foolish
- impious, godless
