# G2754 κενοδοξία

## kenodoxía

_(ken-od-ox-ee'-ah | keh-noh-thoh-KSEE-ah | kay-noh-thoh-KSEE-ah)_

### Definition

from G2755; empty glorying, i.e., self-conceit; vain-glory; 

- vain glory, groundless, self esteem, empty pride
- a vain opinion, error
