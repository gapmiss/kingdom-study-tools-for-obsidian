# G1319 διδασκαλία

## didaskalía

_(did-as-kal-ee'-ah | thee-tha-ska-LEE-ah | thee-tha-ska-LEE-ah)_

### Definition

from G1320; instruction (the function or the information); doctrine, learning, teaching; 

- teaching, instruction
- teaching
  - that which is taught, doctrine
  - teachings, precepts
