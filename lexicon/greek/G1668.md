# G1668 ἕλκος

## hélkos

_(hel'-kos | HEL-kose | ALE-kose)_

### Definition

probably from G1670; an ulcer (as if drawn together); sore; 

- a wound, especially a wound producing a discharge pus
- a sore, an ulcer
