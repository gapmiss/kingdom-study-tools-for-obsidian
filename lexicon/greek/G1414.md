# G1414 δυνατέω

## dynatéō

_(doo-nat-eh'-o | thoo-na-TEH-oh | thyoo-na-TAY-oh)_

### Definition

from G1415; to be efficient (figuratively); be mighty; 

- to be powerful or mighty
- show one's self powerful
- to be able, have power
