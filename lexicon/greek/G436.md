# G436 ἀνθίστημι

## anthístēmi

_(anth-is'-tay-mee | an-THEE-stay-mee | an-THEE-stay-mee)_

### Definition

from G0473 and G2476; to stand against, i.e., oppose; resist, withstand; 

- to set one's self against, to withstand, resist, oppose
- to set against
