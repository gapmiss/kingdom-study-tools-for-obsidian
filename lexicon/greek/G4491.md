# G4491 ῥίζα

## rhíza

_(hrid'-zah | REE-za | REE-za)_

### Definition

apparently a primary word; a "root" (literally or figuratively); root; 

- a root
- that which like a root springs from a root, a sprout, shoot
- metaphorically offspring, progeny
