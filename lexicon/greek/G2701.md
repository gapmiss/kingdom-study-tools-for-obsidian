# G2701 κατατρέχω

## katatréchō

_(kat-at-rekh'-o | ka-ta-TREH-hoh | ka-ta-TRAY-hoh)_

### Definition

from G2596 and G5143; to run down, i.e., hasten from a tower; run down; 

- to run down, hasten down
