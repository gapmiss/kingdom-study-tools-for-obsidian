# G3422 μνημόσυνον

## mnēmósynon

_(mnay-mos'-oo-non | m-nay-MOH-soo-none | m-nay-MOH-syoo-none)_

### Definition

from G3421; a reminder (memorandum), i.e., record; memorial; 

- a memorial (that by which the memory of any person or thing is preserved), a remembrance
