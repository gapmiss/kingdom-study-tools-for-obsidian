# G2028 ἐπονομάζω

## eponomázō

_(ep-on-om-ad'-zo | ep-oh-noh-MA-zoh | ape-oh-noh-MA-zoh)_

### Definition

from G1909 and G3687; to name further, i.e., denominate; call; 

- to put a name upon, name
- to be named
