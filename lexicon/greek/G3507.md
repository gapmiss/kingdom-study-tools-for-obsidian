# G3507 νεφέλη

## nephélē

_(nef-el'-ay | neh-FEH-lay | nay-FAY-lay)_

### Definition

from G3509; properly, cloudiness, i.e., (concretely) a cloud; cloud; 

- a cloud
  - used of the cloud which led the Israelites in the wilderness
