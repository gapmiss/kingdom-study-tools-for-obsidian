# G4063 περιτρέχω

## peritréchō

_(per-ee-trekh'-o | peh-ree-TREH-hoh | pay-ree-TRAY-hoh)_

### Definition

from G4012 and G5143 (including its alternate); to run around, i.e., traverse; run through; 

- to run around, run around about
