# G4890 συνδρομή

## syndromḗ

_(soon-drom-ay' | soon-throh-MAY | syoon-throh-MAY)_

### Definition

from (the alternate of) G4936; a running together, i.e., (riotous) concourse; run together; 

- a running together, concourse, especially hostile or riotous
