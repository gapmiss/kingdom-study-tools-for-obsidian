# G4370 προστρέχω

## prostréchō

_(pros-trekh'-o | prose-TREH-hoh | prose-TRAY-hoh)_

### Definition

from G4314 and G5143 (including its alternate); to run towards, i.e., hasten to meet or join; run (thither to, to); 

- to run to
