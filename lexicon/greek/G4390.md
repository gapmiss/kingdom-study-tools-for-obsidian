# G4390 προτρέχω

## protréchō

_(prot-rekh'-o | proh-TREH-hoh | proh-TRAY-hoh)_

### Definition

from G4253 and G5143 (including its alternate); to run forward, i.e., outstrip, precede; outrun, run before; 

- to run before, to outrun
