# G1538 ἕκαστος

## hékastos

_(hek'-as-tos | HEK-ah-stose | AKE-ah-stose)_

### Definition

as if a superlative of ἕκας (afar); each or every; any, both, each (one), every (man, one, woman), particularly; 

- each, every
