# H7412 רְמָה / רמה

_(rem-aw' | reh-MAW | reh-MA)_

## Definition

corresponding to H7411; to throw, set, (figuratively) assess; cast (down), impose; v

- to cast, throw
  - (P'al)
    - to cast
    - to throw
      - of tribute (figuratively)
  - (Peil)
    - to be cast
    - to be placed, be set
  - (Ithp'el) to be cast
