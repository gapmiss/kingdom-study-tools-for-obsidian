# H428 אֵלֶּה / אלה

## ʼêl-leh

_(ale'-leh | ale-LEH | ay-LEH)_

## Definition

prolonged from H0411; these or those; an-(the) other; one sort, so, some, such, them, these (same), they, this, those, thus, which, who(-m); d

- these
  - used before antecedent
  - used following antecedent
