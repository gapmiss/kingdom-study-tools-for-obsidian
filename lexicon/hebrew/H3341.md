# H3341 יָצַת / יצת

## yâtsath

_(yaw-tsath' | yaw-TSAHT | ya-TSAHT)_

## Definition

a primitive root; to burn or set on fire; figuratively, to desolate; burn (up), be desolate, set (on) fire (fire), kindle; v

- to kindle, burn, set on fire
  - (Qal) to kindle
  - (Niphal)
    - to be kindled
    - to be left desolate
  - (Hiphil) to set on fire, kindle
