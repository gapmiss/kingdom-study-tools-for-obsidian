# H2941 טַעַם / טעם

## ṭaʻam

_(tah'-am | ta-AM | ta-AM)_

## Definition

from H2939 (compare H2940); properly, a taste, i.e., a judicial sentence; account, × to be commanded, commandment, matter; n-m

- taste, judgment, command
  - taste
  - judgment, discretion
  - report
  - command
