# H194 אוּלַי / אולי

## ʼûwlay

_(oo-lah'ee | oo-LAI | oo-LAI)_

## Definition

or (shortened) אֻלַי; from H0176; if not; hence perhaps; if so be, may be, peradventure, unless; adv

- perhaps, peradventure
- if peradventure
- unless
- suppose
