# H1757 דּוּרָא / דורא

## Dûwrâʼ

_(doo-raw' | doo-RAW | doo-RA)_

## Definition

dwelling; probably from H1753; circle or dwelling; Dura, a place in Babylonia; Dura; n-pr-loc

- a place in Babylonia where Nebuchadnezzar set up the golden image, site uncertain
