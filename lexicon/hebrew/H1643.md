# H1643 גֶּרֶשׂ / גרש

## geres

_(gheh'-res | ɡeh-RES | ɡeh-RES)_

## Definition

from an unused root meaning to husk; a kernel (collectively), i.e., grain; beaten corn; n-m

- a crushing (that which is crushed), grain, grits, groats
