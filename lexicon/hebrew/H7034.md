# H7034 קָלָה / קלה

## qâlâh

_(kaw-law' | kaw-LAW | ka-LA)_

## Definition

a primitive root; to be light (as implied in rapid motion), but figuratively, only (be [causatively, hold] in contempt); base, contemn, despise, lightly esteem, set light, seem vile; v

- to disgrace, dishonor, be lightly esteemed, be dishonored, be despised
  - (Niphal) to be dishonored, be despised, be lightly esteemed
  - (Hiphil) to treat with contempt, dishonor, treat shamefully
