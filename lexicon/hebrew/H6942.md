# H6942 קָדַשׁ / קדש

## qâdash

_(kaw-dash' | kaw-DAHSH | ka-DAHSH)_

## Definition

a primitive root; to be (causatively, make, pronounce or observe as) clean (ceremonially or morally); appoint, bid, consecrate, dedicate, defile, hallow, (be, keep) holy(-er) (place), keep, prepare, proclaim, purify, sanctify(-ied one) (self), × wholly; v

- to consecrate, sanctify, prepare, dedicate, be hallowed, be holy, be sanctified, be separate
  - (Qal)
    - to be set apart, be consecrated
    - to be hallowed
    - consecrated, tabooed
  - (Niphal)
    - to show oneself sacred or majestic
    - to be honored, be treated as sacred
    - to be holy
  - (Piel)
    - to set apart as sacred, consecrate, dedicate
    - to observe as holy, keep sacred
    - to honor as sacred, hallow
    - to consecrate
  - (Pual)
    - to be consecrated
    - consecrated, dedicated
  - (Hiphil)
    - to set apart, devote, consecrate
    - to regard or treat as sacred or hallow
    - to consecrate
  - (Hithpael)
    - to keep oneself apart or separate
    - to cause Himself to be hallowed (of God)
    - to be observed as holy
    - to consecrate oneself
