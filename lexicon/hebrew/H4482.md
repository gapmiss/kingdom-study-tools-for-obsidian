# H4482 מֵן / מן

## mên

_(mane | mane | mane)_

## Definition

from an unused root meaning to apportion; a part; hence, a musical chord (as parted into strings); in [the same] (Psalm 68:23), stringed instrument (Psalm 150:4), whereby (Psalm 45:8 [defective plural]); n-m

- string (of harp)
- portion
