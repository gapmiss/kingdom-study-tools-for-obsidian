# H5646 עָב / עב

## ʻâb

_(awb | awb | av)_

## Definition

or עֹב; from an unused root meaning to cover; properly, equivalent to H5645; but used only as an architectural term; an architrave (as shading the pillars); thick (beam, plant); n-m

- an architectural term
  - meaning dubious; perhaps projecting roof, landing, thick beam, plank, threshold
