# H7076 קִנָּמוֹן / קנמון

## qinnâmôwn

_(kin-naw-mone' | kin-naw-MONE | kee-na-MONE)_

## Definition

from an unused root (meaning to erect); cinnamon bark (as in upright rolls); cinnamon; n-m

- cinnamon
  - fragrant bark used as spice
