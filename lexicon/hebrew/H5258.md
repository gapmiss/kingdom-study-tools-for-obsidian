# H5258 נָסַךְ / נסך

## nâçak

_(naw-sak' | naw-SAHK | na-SAHK)_

## Definition

a primitive root; to pour out, especially a libation, or to cast (metal); by analogy, to anoint a king; cover, melt, offer, (cause to) pour (out), set (up); v

- to pour out, pour, offer, cast
  - (Qal)
    - to pour out
    - to cast metal images
    - to anoint (a king)
  - (Niphal) to be anointed
  - (Piel) to pour out (as a libation)
  - (Hiphil) to pour out libations
  - (Hophal) to be poured out
- to set, install
  - (Qal) to install
  - (Niphal) to be installed
