# H8427 תָּוָה / תוה

## tâvâh

_(taw-vaw' | taw-WAW | ta-VA)_

## Definition

a primitive root; to mark out, i.e., (primitive) scratch or (definite) imprint; scrabble, set (a mark); v

- to scrabble, limit, mark, make or set a mark
  - (Piel) to mark
  - (Hiphil) to set a mark
