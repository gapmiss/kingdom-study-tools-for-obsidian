# H1847 דַּעַת / דעת

## daʻath

_(dah'-ath | da-AT | da-AT)_

## Definition

from H3045; knowledge; cunning, (ig-)norantly, know(-ledge), (un-)awares (wittingly); n

- knowledge
  - knowledge, perception, skill
  - discernment, understanding, wisdom
