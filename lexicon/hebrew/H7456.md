# H7456 רָעֵב / רעב

## râʻêb

_(raw-abe' | raw-ABE | ra-AVE)_

## Definition

a primitive root; to hunger; (suffer to) famish, (be, have, suffer, suffer to) hunger(-ry); v

- to be hungry, be voracious
  - (Qal) to be hungry
  - (Hiphil) to allow one to be hungry, allow to hunger
