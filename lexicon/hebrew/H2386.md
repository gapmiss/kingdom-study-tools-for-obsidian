# H2386 חֲזִיר / חזיר

## chăzîyr

_(khaz-eer' | huh-DZEER | huh-ZEER)_

## Definition

from an unused root probably meaning to inclose; a hog (perhaps as penned); boar, swine; n-m

- hog, swine, boar
  - swine (as forbidden food)
  - wild boar
