# H5459 סְגֻלָּה / סגלה

## çᵉgullâh

_(seg-ool-law' | seh-ɡool-LAW | seh-ɡoo-LA)_

## Definition

feminine passive participle of an unused root meaning to shut up; wealth (as closely shut up); jewel, peculiar (treasure), proper good, special; n-f

- possession, property
  - valued property, peculiar treasure
  - treasure
