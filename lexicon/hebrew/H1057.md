# H1057 בָּכָא / בכא

## bâkâʼ

_(baw-kaw' | baw-KAW | ba-HA)_

## Definition

the same as H1056; the weeping tree (some gum-distilling tree, perhaps the balsam); mulberry tree; n-m

- balsam tree—a shrub which drips sap when it is cut
