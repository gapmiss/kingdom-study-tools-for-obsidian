# H6835 צַפַּחַת / צפחת

## tsappachath

_(tsap-pakh'-ath | tsahp-pa-HAHT | tsa-pa-HAHT)_

## Definition

from an unused root meaning to expand; a saucer (as flat); cruse; n-f

- jar, jug (for liquids)
