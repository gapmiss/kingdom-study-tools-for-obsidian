# H948 בּוּץ / בוץ

## bûwts

_(boots | boots | boots)_

## Definition

from an unused root (of the same form) meaning to bleach, i.e., (intransitive) be white; probably cotton (of some sort); fine (white) linen; n-m

- byssus, a costly, fine white linen cloth made in Egypt
