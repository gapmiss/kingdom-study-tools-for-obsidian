# H6903 קְבֵל / קבל

## qᵉbêl

_(keb-ale' | keh-BALE | keh-VALE)_

## Definition

or קֳבֵל; (Aramaic), (corresponding to H6905; (adverbially) in front of; usually (with other particles) on account of, so as, since, hence; + according to, + as, + because, before, + for this cause, + forasmuch as, + by this means, over against, by reason of, + that, + therefore, + though, + wherefore; conj

- (subst) front
- (prep) in front of, before, because of, in view of, by reason of, because of this, therefore
- (conj) because that, inasmuch as, although, according as, before
- (adv) accordingly, then
