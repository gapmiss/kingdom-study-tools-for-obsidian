# H5708 עֵד / עד

## ʻêd

_(ayd | ade | ade)_

## Definition

from an unused root meaning to set a period (compare H5710, H5749); the menstrual flux (as periodical); by implication (in plural) soiling; filthy; n-f

- menstruation
  - filthy rag, stained garment (figurative for best deeds of guilty people)
