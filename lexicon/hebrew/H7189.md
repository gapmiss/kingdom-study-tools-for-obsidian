# H7189 קֹשֶׁט / קשט

## qôsheṭ

_(ko'-shet | koh-SHET | koh-SHET)_

## Definition

or קֹשְׁטְ; from an unused root meaning to balance; equity (as evenly weighed), i.e., reality; truth; n-m

- bow
- truth, balanced verity
