# H5183 נַחַת / נחת

## Nachath

_(nakh'-ath | na-HAHT | na-HAHT)_

## Definition

from H5182; a descent, i.e., imposition, unfavorable (punishment) or favorable (food); lighting down, quiet(-ness), to rest, be set on; n

- rest, quietness
  - quietness, quiet attitude
  - rest (of death)
- descent (of judgment)
