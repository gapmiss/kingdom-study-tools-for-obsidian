# H3863 לוּא / לוא

## lûwʼ

_(loo | loo | loo)_

## Definition

or לֻא; or לוּ; a conditional particle; if; by implication (interj. as a wish) would that!; if (haply), peradventure, I pray thee, though, I would, would God (that); conj

- if, oh that!, if only!
  - if (unlikely condition)
  - if only!, oh that!, would that!
