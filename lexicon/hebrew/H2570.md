# H2570 חֹמֶשׁ / חמש

## chômesh

_(kho'-mesh | hoh-MESH | hoh-MESH)_

## Definition

from an unused root probably meaning, to be stout; the abdomen (as obese); fifth (rib); n-m

- belly, abdomen, fifth ribs
