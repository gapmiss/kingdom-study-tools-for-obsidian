# H6343 פַּחַד / פחד

## pachad

_(pakh'-ad | pa-HAHD | pa-HAHD)_

## Definition

from H6342; a (sudden) alarm (properly, the object feared, by implication, the feeling); dread(-ful), fear, (thing) great(-ly feared) (fear), terror; n-m

- terror, dread
  - dread
  - object of dread
