# H3649 כָּמָר / כמר

## kâmâr

_(kaw-mawr' | kaw-MAWR | ka-MAHR)_

## Definition

from H3648; properly, an ascetic (as if shrunk with self-maceration), i.e., an idolatrous priest (only in plural); Chemarims (idolatrous) priests; n-m

- priest, idolatrous priest
