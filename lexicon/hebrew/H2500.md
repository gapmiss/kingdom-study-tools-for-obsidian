# H2500 חֵלֶף / חלף

## chêleph

_(klay'-lef | hay-LEP | hay-LEF)_

## Definition

from H2498; properly, exchange; hence (as preposition) instead of; × for; n-m

- (n m) in exchange for (only construct)
- (prep) in return for
