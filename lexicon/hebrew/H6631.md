# H6631 צֶאֱצָא / צאצא

## tseʼĕtsâʼ

_(tseh-ets-aw' | tseh-eh-TSAW | tseh-ay-TSA)_

## Definition

from H3318; issue, i.e., produce, children; that which cometh forth (out), offspring; n-m

- offspring, produce, issue
  - offspring (of men)
  - produce (of earth)
  - descendants (metaphorical)
