# H4371 מֶכֶס / מכס

## mekeç

_(meh'-kes | meh-KES | meh-HES)_

## Definition

probably from an unused root meaning to enumerate; an assessment (as based upon a census); tribute; n-m

- computation, proportion to be paid, tribute, tax
