# H6189 עָרֵל / ערל

## ʻârêl

_(aw-rale' | aw-RALE | ah-RALE)_

## Definition

from H6188; properly, exposed, i.e., projecting loose (as to the prepuce); used only technically, uncircumcised (i.e., still having the prepuce uncurtailed); uncircumcised (person); n-m

- uncircumcised, having foreskin
