# H4219 מִזְרָק / מזרק

## mizrâq

_(miz-rawk' | midz-RAWK | meez-RAHK)_

## Definition

from H2236; a bowl (as if for sprinkling); bason, bowl; n-m

- bowl, basin
  - bowl (for wine)
  - basin (vessel for throwing or tossing a liquid)
