# H46 אָבִיר / אביר

## ʼâbîyr

_(aw-beer' | aw-BEER | ah-VEER)_

## Definition

from H0082; mighty (spoken of God); mighty (one); a-m

- strong, mighty—used only to describe God
- the Strong—old name for God (poetic)
