# H7326 רוּשׁ / רוש

## rûwsh

_(roosh | roosh | roosh)_

## Definition

a primitive root; to be destitute; lack, needy, (make self) poor (man); v

- to be poor, be in want, lack
  - (Qal)
    - to be poor
    - to be in want or hunger
    - poor man (subst)
  - (Hithpolel) one impoverishing himself (participle)
