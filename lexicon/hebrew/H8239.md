# H8239 שָׁפַת / שפת

## shâphath

_(shaw-fath' | shaw-PAHT | sha-FAHT)_

## Definition

a primitive root; to locate, i.e., (generally) hang on or (figuratively) establish, reduce; bring, ordain, set on; v

- to set, place, put, ordain
  - (Qal)
    - to set
    - to ordain, establish
