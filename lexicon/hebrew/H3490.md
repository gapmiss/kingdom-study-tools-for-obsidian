# H3490 יָתוֹם / יתום

## yâthôwm

_(yaw-thome' | yaw-TOME | ya-TOME)_

## Definition

from an unused root meaning to be lonely; a bereaved person; fatherless (child), orphan; n-m

- an orphan, fatherless
