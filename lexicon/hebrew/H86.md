# H86 אַבְרֵךְ / אברך

## ʼabrêk

_(ab-rake' | ab-RAKE | av-RAKE)_

## Definition

probably an Egyptian word meaning; kneel; bow the knee; v

- a shout made to announce Joseph's chariot
  - command
  - bow

## Note
