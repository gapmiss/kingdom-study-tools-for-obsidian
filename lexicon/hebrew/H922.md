# H922 בֹּהוּ / בהו

## bôhûw

_(bo'-hoo | boh-HOO | boh-HOO)_

## Definition

from an unused root (meaning to be empty); a vacuity, i.e., (superficially) an undistinguishable ruin; emptiness, void; n-m

- emptiness, void, waste
